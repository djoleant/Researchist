## 📦 Stack
* Neo4jClient
* Asp.net WebApi
* React (JS)

## 🚀 Quickstart

```bash
> # Clone repository
> git clone https://github.com/djoleant/Researchist.git
> cd Researchist

> # Server start
> cd Researchist/Researchist
> dotnet watch run

> # Client start
> cd ../../ResearchistApp
> npm install
> npm start

> # Start neo4j
> bin/neo4j start
```


## ⚙️ Configuration options

| Environment variable name | Default value | 
| ------------------------- | ------------- |
| PORT | 7687 | 
| NEO4J_CLIENT | BoltGraphClient |
| NEO4J_USER | neo4j |
| NEO4J_PASSWORD | researchist |
| NEO4J_DATABASE | neo4j |
| NEO4J_VERSION | 5.2 |
